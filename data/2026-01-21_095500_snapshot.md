## Formula Reference

The formula syntax uses these prefixes:
- `V{groupId}.{inputIndex}` - Reference input value (e.g., V1.1 = Group 1, Input 1)
- `V{groupId}` - Sum of all inputs in group
- `C{groupId}.{inputIndex}` - Constant value
- `L{groupId}.{inputIndex}` - Lookup value
- `S{groupId}` - Series sum
- `F{keyPeriodId}` - Flag for key period (1 during period, 0 otherwise)
- `I{indexId}` - Indexation factor
- `R{calcId}` - Reference another calculation result
- `T.HiY` - Hours in year, `T.MiY` - Minutes in year, `T.DiM` - Days in month
